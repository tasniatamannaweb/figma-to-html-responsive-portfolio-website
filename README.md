# figma-to-html-responsive-portfolio-website
Responsive website converted from Figma design using HTML, CSS, Bootstrap, and custom media queries.
A fully responsive website converted from a Figma UI design using semantic HTML, Bootstrap framework, and custom CSS media queries without JavaScript.

• HTML5
• CSS3
• Bootstrap 5
• Custom Media Queries
